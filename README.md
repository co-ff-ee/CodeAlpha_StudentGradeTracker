# Student Grade Tracker (Java)

## Description
A console-based Java application to manage and analyze student grades.
The program allows users to add student details and calculates average,
highest, and lowest marks.

## Features
- Add student records
- Calculate average marks
- Find highest and lowest marks
- Console-based interface

## Technologies Used
- Java
- OOP Concepts
- Collections (ArrayList)

## How to Run
1. Clone the repository
2. Compile and run Main.java
